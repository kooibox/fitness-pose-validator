## 评估指标说明

| 指标 | 说明 | 目标值 |
|-----|------|-------|
| 计数准确率 | 正确计数/真实次数 | >95% |
| 抖动系数 | 静止时角度波动 | <5° |
| 误触发 | 多余的状态切换 | 尽量少 |
